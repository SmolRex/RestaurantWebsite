# 🍷 Royce & Co. - A Fine Dining Experience in Code

**Project Showcase #1: Restaurant Website Demo**  
*Frontend with animations + Reservation Feature*

---

## 📋 About The Project

An elegant **FICTIONAL** restaurant website concept built from scratch with HTML, CSS & JavaScript.

### ✨ Key Features

- **Smooth scrolling animations** & logo transitions
- **Curated menu pages**: Signatures, À La Carte, Mains, Wines, Cocktails
- **Interactive reservation system** with customer list
- **Immersive design** inspired by fine dining aesthetics
- Fully responsive and modern UI

---

- HTML5
- CSS3
- JavaScript (Vanilla)
